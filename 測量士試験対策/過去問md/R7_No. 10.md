**〔No.10〕 GNSS基準点測量**

**問題：** #GNSS基準点測量 について、明らかに間違っているものはどれか。 

**回答時の選択肢：** 

| 番号    | 内容                                        |
| :---- | :---------------------------------------- |
| 1     | 異なる機種のアンテナを組み合わせた測量では、原則としてPCV補正が必要。      |
| 2     | 時計のずれによる誤差は、二重位相差による解析処理で消去できる。           |
| 3     | スタティック法は、同時に信号を受信し、観測点間の基線ベクトルを求める。       |
| **4** | **スタティック法では、GNSS衛星の軌道情報に精密暦を用いなければならない。** |
| 5     | ネットワーク型RTK法は、補正データを通信回線で受信し即時に位置を求める。     |

**正解： 4**
**解説：** 準則第41条により、軌道情報は「放送暦を標準」としており、精密暦は必須ではありません。
