**〔No.6〕 測量の基準**

**問題：** 測量法における #測量の基準 について、明らかに間違っているものはどれか。 

**回答時の選択肢：** 

| 番号    | 内容                               |
| :---- | :------------------------------- |
| 1     | 位置は、地理学的経緯度及び平均海面からの高さで表示する。     |
| **2** | **面積は、ジオイド上の値で表示する。**            |
| 3     | 回転楕円体は、中心が地球の重心と一致し、短軸が自転軸と一致する。 |
| 4     | 地理学的経緯度は、世界測地系に従って測定しなければならない。   |
| 5     | 測量の原点は、日本経緯度原点及び日本水準原点とする。       |

**正解： 2** 

**解説：** 測量法第11条により、距離および面積は「回転楕円体の表面上の値」で表示しなければなりません。
