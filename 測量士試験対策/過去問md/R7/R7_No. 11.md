**〔No.11〕 標高計算（ #ジオイド ）**

**問題：** ジオイド傾斜（1kmにつき+0.05m）がある地域で、新点A〜Eのうち最も標高が高い点はどれか。

| 点名    | 距離      | 標高     | 楕円体高   |
| :---- | :------ | ------ | ------ |
| A     | 0 m     | 50.00m | 91.40m |
| B     | 2,000 m |        | 91.60m |
| C     | 4,000 m |        | 91.65m |
| **D** | 6,000 m |        | 91.70m |
| E     | 8,000 m |        | 91.75m |

• 点A：楕円体高 91.40m, 標高 50.00m
• 点B：2,000m地点, 楕円体高 91.60m
• 点C：4,000m地点, 楕円体高 91.65m
• 点D：6,000m地点, 楕円体高 91.70m
• 点E：8,000m地点, 楕円体高 91.75m

**正解： 2 (新点B)** **解説：** 各地点のジオイド高を計算し、楕円体高から差し引いた標高を比較すると、B点が最高となります。
