**〔No.1〕 測量法（定義・登録）**

**問題：** #測量法 に規定された事項について、明らかに間違っているものだけの組合せはどれか。

• a. 「 #基本測量 」とは、全ての測量の基礎となる測量で、国が計画する全ての測量が含まれる。
• b. 「 #測量成果 」とは、当該測量において最終の目的として得た結果をいい、「 #測量記録 」とは、作業記録をいう。
• c. 公共測量を実施する者は、関係市町村長に対し必要な情報の提供を求めることができる。
• d. #測量計画機関 は、自ら #測量作業機関 となることはできない。
• e. 「 #測量業 」とは、基本測量、公共測量又はそれら以外の測量を請け負う営業をいう。

**回答時の選択肢：** 

| 番号    | 組合せ      |
| :---- | :------- |
| 1     | a, c     |
| **2** | **a, d** |
| 3     | b, d     |
| 4     | b, e     |
| 5     | c, e     |

**正解： 2** 

**解説：** 
aは、基本測量は「国土地理院の長が実施するもの」に限定されます（法第4条）。
dは、測量計画機関が自ら実施する場合は測量作業機関になれます（法第7条）。
