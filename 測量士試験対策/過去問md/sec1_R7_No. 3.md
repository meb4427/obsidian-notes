**〔No.3〕 #公共測量 の諸手続**

**問題：** 公共測量における機関の対応として、明らかに間違っているものはどれか。 

**回答時の選択肢：** 

|番号 | 内容 |
| :--- | :--- |
| **1** | **技術的実績があることから、計画書について国土地理院の長への提出を省略した。** |
| 2 | 財産権や個人情報保護などに関する法令を遵守して作業を行った。 |
| 3 | 得ようとする測量成果の種類や品質等を示す製品仕様書を定めた。 |
| 4 | 「地理情報標準プロファイル（JPGIS）」に準拠して製品仕様書を定めた。 |
| 5 | 作業着手前に適切な作業計画を立案し、測量計画機関に提出して承認を得た。 |

**正解： 1**
**解説：**
測量法第36条により、実績に関わらず計画書の提出と技術的助言の取得は必須です。
