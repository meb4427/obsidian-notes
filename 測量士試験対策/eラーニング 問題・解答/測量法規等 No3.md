# 測量法規等 No.03 解答と解説

## 問A 問題

**問A** 次のa〜eの文は、測量法（昭和24年法律第188号）に規定された事項について述べたものである。明らかに間違っているものだけの組合せはどれか。次の1〜5の中から選べ。

a. 「基本測量及び公共測量以外の測量」を実施しようとする者は、あらかじめ、国土交通省令で定めるところにより、その旨を国土交通大臣に届け出なければならない。

b. 測量士又は測量士補となる資格を有する者は、測量士又は測量士補になろうとする場合においては、国土地理院の長に対してその資格を証する書類を添えて、測量士名簿又は測量士補名簿に登録の申請をしなければならない。

c. 公共測量の測量成果を使用して測量を実施しようとする者は、あらかじめ、当該測量成果を得た測量作業機関の承認を得なければならない。

d. 「基本測量及び公共測量以外の測量」とは、基本測量及び公共測量以外の測量成果を使用して実施する測量をいう。

e. 測量標とは、永久標識、一時標識及び仮設標識をいう。

1. a, b
    
2. a, c, d
    
3. b, e
    
4. c, d
    
5. c, d, e
    
---

## 問A 解説

**【測量士】** **問A** **［解答］** 4. c, d

**［解説］** **a. 「届出等」（法律46条）に関する問題である。** 法律46条には、「基本測量及び公共測量以外の測量を実施しようとする者は、あらかじめ、国土交通省令で定めるところにより、その旨を国土交通大臣に届け出なければならない。」と規定しているので、aの文は正しい。
[[測量法#第四十六条（届出等） 02EL]]

**b. 「測量士及び測量士補の登録」（法律49条）に関する問題である。** 法律49条には、「次条又は第51条の規定により測量士又は測量士補となる資格を有する者は、測量士又は測量士補になろうとする場合においては、国土地理院の長に対してその資格を証する書類を添えて、測量士名簿又は測量士補名簿に登録の申請をしなければならない。」と規定しているので、bの文は正しい。
[[測量法#第四十九条（測量士及び測量士補の登録） 02EL]]]

**c. 「測量成果の使用」（法律44条）に関する問題である。** 法律44条には、「公共測量の測量成果を使用して測量を実施しようとする者は、あらかじめ、当該測量成果を得た**測量計画機関**の承認を得なければならない。」と規定している。cの文は、「測量計画機関」が「測量作業機関」と書いてあるので、間違っている。
[[測量法#第四十四条（測量成果の使用） 02EL]]

**d. 「基本測量及び公共測量以外の測量」（法律6条）に関する問題である。** 法律6条には、「この法律において『基本測量及び公共測量以外の測量』とは、基本測量又は公共測量の測量成果を使用して実施する基本測量及び公共測量以外の測量（**建物に関する測量その他の局地的測量又は小縮尺図の調製その他の高度の精度を必要としない測量で政令で定めるものを除く。**）をいう。」と規定している。dの文は、「基本測量及び公共測量以外の測量成果を使用して実施する測量をいう。」と書かれているので、間違っている。
[[測量法#第六条（基本測量及び公共測量以外の測量） 02EL]]

**e. 「測量標」（法律10条）に関する問題である。** 法律10条には、「この法律において『測量標』とは、永久標識、一時標識及び仮設標識をいい、これらは、左の各号に掲げるとおりとする。」
[[測量法#第十条（測量標） 02EL]]

一、二、三、2、3（省略）

と規定しているので、eの文は正しい。

したがって、正解は4である。

## 問B 問題

**問B** 次のa〜eの文は、我が国における測量の基準について述べたものである。明らかに間違っているものだけの組合せはどれか。次の1〜5の中から選べ。

a. 基本測量及び公共測量において、位置は、地理学的経緯度及び楕円体高で表示する。ただし、場合により、垂直距離及び平均海面からの高さ、極座標及び平均海面からの高さ又は地心直交座標で表示することができる。

b. 公共測量における距離及び面積は、ジオイドの表面上の値で表示する。

c. 測量の原点は、離島の測量等の場合で国土地理院の長の承認を得たときを除き、日本経緯度原点及び日本水準原点とする。

d. 回転楕円体の長半径及び扁平率の値は政令で定められており、過去にこれらの値が改正されたことがある。

e. 平面直角座標（平成14年国土交通省告示第9号）は、座標系のX軸を座標系原点において子午線に一致する軸とし、座標系のX軸を座標系原点において、座標系のX軸に直交する軸とする。

1. a, b, c
    
2. a, b, d
    
3. a, b, e
    
4. b, c, d
    
5. b, d, e
    

---

## 問B 解説

**問B** **［解答］** 3. a, b, e

**［解説］** **a. 「測量の基準」（法律11条）に関する問題である。** 法律11条には、「基本測量及び公共測量は、次に掲げる測量の基準に従って行わなければならない。

一 位置は、地理学的経緯度及び平均海面からの高さで表示する。ただし、場合により、直角座標及び平均海面からの高さ、極座標及び平均海面からの高さ又は地心直交座標で表示することができる。

二、三、四（省略）」と規定している。aの文は、「楕円体高で表示する。」と書いているので、間違っている。

**b. 「測量の基準」（法律11条）に関する問題である。** 法律11条には、「基本測量及び公共測量は、次に掲げる測量の基準に従って行わなければならない。

（省略）

二 距離及び面積は、前三項に規定する**回転楕円体の表面上の値**で表示する。」と規定している。bの文は、「ジオイドの表面上」と書いてあるので、間違っている。

**c. 「測量の基準」（法律11条）に関する問題である。** 法律11条には、「基本測量及び公共測量は、次に掲げる測量の基準に従って行わなければならない。

一 位置は、地理学的経緯度及び平均海面からの高さで表示する。ただし、場合により、直角座標及び平均海面からの高さ、極座標及び平均海面からの高さ又は地心直交座標で表示することができる。

二（省略）

三 測量の原点は、日本経緯度原点及び日本水準原点とする。ただし、離島の測量その他特別の事情がある場合において、国土地理院の長の承認を得たときは、この限りでない。

四（省略）」と規定している。cの文はこれと同じ内容なので、正しい。

**d. 「測量の基準」（法律11条）に関する問題である。** 法律11条には、「基本測量及び公共測量は、次に掲げる測量の基準に従って行わなければならない。

一、二、三（省略）

2 前項第一号の地理学的経緯度は、世界測地系に従って測定しなければならない。

3 前項の「世界測地系」とは、地球を次に掲げる要件を満たす扁平な回転楕円体であると想定して行う地理学的経緯度の測定に関する測量の基準をいう。

一 その長半径及び扁平率が、地理学的経緯度の測定に関する国際的な決定に基づき政令で定める値であるものであること。

二、三（省略）」と規定している。回転楕円体の長半径と扁平率の改正については、現在、世界測地系ではGRS80楕円体（長半径及び扁平率は6378137m及び298.257222101分の1）が採用されているが、世界測地系導入前はベッセル楕円体（長半径及び扁平率は6377397.155m及び299.152813分の1）が採用されていた。したがって、dの文は、正しい。

**e. 「平面直角座標系」（平成14年国土交通省告示第9号）に関する問題である。** 平成14年国土交通省告示第9号の備考には、「座標系のX軸は、座標系原点において子午線に一致する軸とし、真北に向かう値を正とし、座標系のY軸は、座標系原点において座標系のX軸に直交する軸とし、真東に向かう値を正とする。」と規定している。eの文はX軸とY軸の記述がこの規定と逆であり、間違っている。

したがって、正解は3である。

## 問C 問題

**問C** 次の文は、正規分布の性質について述べたものである。 [ ア ] 〜 [ ウ ] に入る語句の組合せとして最も適当なものはどれか。次の1〜5の中から選べ。

なお、関数の値が必要な場合は、[関数表](https://www.google.com/search?q=%23)を使用すること。

確率変数が平均 $\mu$ 、分散 $\sigma^2$ の正規分布に従うとき、$x \sim N(\mu, \sigma^2)$ と書くこととする。二つの互いに独立な確率変数 $x_1, x_2$ について、

$x_1 \sim N(\mu_1, \sigma_1^2)$

$x_2 \sim N(\mu_2, \sigma_2^2)$

のとき、

$ax_1 + bx_2 \sim N(a\mu_1 + b\mu_2, a^2\sigma_1^2 + b^2\sigma_2^2)$

となることが知られている。ただし、 $a, b$ は任意の実数である。

いま、異なる2種類のトータルステーションA、Bを用いて、ある2点間の距離を多数回測定し、気象補正を施したところ、それぞれ距離の測定値 $x_A, x_B$ の分布は、

$x_A \sim N(200.006, 0.000004)$

$x_B \sim N(200.004, 0.000008)$

となることがわかったとする。

ただし、$x_A$ と $x_B$ は互いに独立で、それぞれの分布は偶然誤差によって生じるものとし、その他の観測条件は同一であるものとする。なお、距離の単位はmである。

このとき、$x_A, x_B$ の平均となる確率変数 $x_P$ は、

$x_P = \frac{x_A + x_B}{2} \sim N([ ア ], [ イ ])$

となり、標準偏差は [ ウ ] である。

||**ア**|**イ**|**ウ**|
|---|---|---|---|
|1.|200.005|0.000006|0.0024|
|2.|200.005|0.000003|0.0017|
|3.|200.005|0.000005|0.0019|
|4.|200.006|0.000006|0.0021|
|5.|200.006|0.000003|0.0024|

---

## 問C 解説

**問C** **［解答］** 2

**［解説］** 正規分布、誤差伝播に関する問題である。

$x_A \sim N(200.006, 0.000004)$

$x_B \sim N(200.004, 0.000008)$

であるとき、

$x_P = \frac{x_A + x_B}{2}$ の式で計算される値については、

$a = \frac{1}{2}, b = \frac{1}{2}, \mu_A = 200.006, \mu_B = 200.004, \sigma_A^2 = 0.000004, \sigma_B^2 = 0.000008$ と置くと、

その平均値と分散は、

$a\mu_A + b\mu_B = 200.005$

$a^2\sigma_A^2 + b^2\sigma_B^2 = 0.000003$

となるので、

$\frac{x_A + x_B}{2} \sim N(200.005, 0.000003)$

と書ける。

標準偏差は分散の平方根であるので、 $\sqrt{0.000003} \approx 0.0017$ となる。

したがって、正解は2である。
