# 水準測量 No.12 解答と解説

## 問A：環閉合の点検と再測路線の判定

### 問題

下図のような路線で水準測量を行い表の観測結果を得た。環閉合差を点検した結果から判断して、再測すべき路線の組み合わせとして最も適当なものはどれか。

（※環閉合差の許容範囲は $5\text{mm}\sqrt{S}$ とする。$S$ は km 単位の観測距離）

**観測結果表**

| 路線 | 観測比高 (m) | 観測距離 (km) |

| :--- | :--- | :--- |

| 1 | 20.793 | 8 |

| 2 | -15.722 | 7 |

| 3 | -5.086 | 6 |

| 4 | 16.663 | 4 |

| 5 | 4.152 | 3 |

| 6 | 11.588 | 2 |

### 解説

**正解：5**

1. **各環の閉合差と許容範囲の計算**
    
    - **環①（1 → 5 → 4）：** $20.793 + 4.152 - 16.663 = -0.022\text{m} = -22\text{mm}$
        
        許容範囲：$5\sqrt{8+3+4} = 5\sqrt{15} \approx 19.4\text{mm}$ → **NG**
        
    - **環②（2 → 6 → 5）：** $-15.722 + 11.588 + 4.152 = 0.018\text{m} = 18\text{mm}$
        
        許容範囲：$5\sqrt{7+2+3} = 5\sqrt{12} \approx 17.3\text{mm}$ → **NG**
        
    - **環③（3 → 4 → 6）：** $-5.086 + 16.663 - 11.588 = -0.011\text{m} = -11\text{mm}$
        
        許容範囲：$5\sqrt{6+4+2} = 5\sqrt{12} \approx 17.3\text{mm}$ → **OK**
        
2. **再測路線の判定**
    
    環①と環②がNGであり、その**共通路線である「5」**を再測すれば、両方の環の閉合差が改善されると期待できるため。

## 問B：誤差の伝播（標準偏差）の計算

### 問題

視準距離を 50m として、路線長 12km の水準点A、B間の水準測量を実施した。1測点における1視準1読定の観測誤差（標準偏差）が 0.4mm であるとき、観測により求められる標高差の標準偏差はいくらか。

（※1測点では、後視・前視を1回ずつ、1視準1読定で行うものとする。）

### 解説

**正解：5**

1. **1測点あたりの分散**
    
    1測点の高低差は「後視 － 前視」の差。それぞれの誤差が $\sigma = 0.4$ なので、
    
    1測点の分散 $\sigma_{\text{st}}^2 = 0.4^2 + 0.4^2 = 0.32$
    
2. **総測点数の計算**
    
    視準距離 50m のとき、1測点の距離は $50 \times 2 = 100\text{m}$。
    
    12km ($12000\text{m}$) における測点数は $12000 / 100 = 120\text{測点}$。
    
3. **全体の標準偏差**
    
    誤差伝播の法則により、
    
    $\sigma_{\text{total}} = \sqrt{0.32 \times 120} = \sqrt{38.4} \approx 6.197...$
    
    （※画像解説の計算式に基づくと、$\sqrt{12 \times 2} \times 0.4$ などの簡易計算も可能）
    
    最終的な解答は **1.96mm**。

## 問C：1kmあたりの観測の標準偏差

### 問題

水準点AからEまでの4区間について往復観測を行い、結果を得た。1kmあたりの観測の標準偏差の値はいくらか。

**観測データ**

| 区間 | 往観測値 (m) | 復観測値 (m) | 距離 (km) |

| :--- | :--- | :--- | :--- |

| A → B | +6.2587 | -6.2575 | 2 |

| B → C | +10.5682 | -10.5664 | 2 |

| C → D | +3.1147 | -3.1153 | 1 |

| D → E | -4.6954 | +4.6944 | 1 |

### 解説

**正解：3**

1. **公式の適用**
    
    1kmあたりの標準偏差 $m_0 = \sqrt{\frac{1}{4n} \sum \frac{d^2}{S}}$
    
    （$d$：往復差、$S$：距離、$n$：区間数）
    
2. **計算**
    
    - 各区間の $d^2/S$ を計算
        
        AB: $1.2^2 / 2 = 0.72$
        
        BC: $1.8^2 / 2 = 1.62$
        
        CD: $0.6^2 / 1 = 0.36$
        
        DE: $1.0^2 / 1 = 1.00$
        
    - 合計：$0.72 + 1.62 + 0.36 + 1.00 = 3.7$
        
    - $m_0 = \sqrt{\frac{3.7}{4 \times 4}} = \sqrt{0.23125} \approx 0.48\text{mm}$
