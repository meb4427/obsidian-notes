# 水準測量 No.11 解答と解説

## 問A：水準測量における誤差と対策

### 問題

次は、水準測量における誤差について述べたものである。明らかに間違っているものはどれか。

1. 渡海水準測量で両岸から同時観測をすることは、気流による影響を軽減することができる。
    
2. 標尺が鉛直に立てられていないために生じる誤差は、標尺付属水準器を使用したり、標尺を前後に傾け読定値が最小となるところを読むことによって軽減できる。
    
3. 地球の曲率によって生じる誤差を軽減するため、前視・後視の視準距離を等しくすることで消去できる。
    
4. レベルの鉛直軸が傾いているために生じる誤差を鉛直軸誤差といい、前後の標尺を結ぶ直線上にレベルを置き、三脚の特定の1本を常に同一の標尺に対向するように整置し観測することで小さくすることができる。
    
5. 地表面近くに吹く気温が著しく入り交じり密度が変化することより視準線が屈折し生じる誤差を大気の屈折誤差といい、傾斜地においては視準距離を短くし、標尺の上方付近での観測を避けることにより小さくすることができる。
    

### 解説

**正解：5**

1. **○：** 大気の温度変化による屈折の影響を相殺するため、両岸から同時観測を行う。
    
2. **○：** 標尺の傾きによる誤差は、最小値（鉛直状態）を読み取ることで軽減できる。
    
3. **○：** 球差（地球の曲率）は、前後視の距離を等しくすれば相殺される。
    
4. **○：** 鉛直軸誤差の対策として、三脚の特定の1本を常に同じ方向に向ける「1脚対向」が有効。
    
5. **×：** **「大気の屈折誤差」ではなく「気流の影響（かげろう）」**についての説明である。大気の屈折誤差（気差）は、地表付近の密度勾配により視準線が湾曲する現象を指し、これも前後視の距離を等しくすることで消去できる。

## 問B：水準測量の誤差と対策（組み合わせ）

### 問題

次のA〜Eは水準測量における誤差である。その対策法としてア〜オにあげた方法をとるとき、最も適当な組み合わせを1〜5の中から選べ。

**【誤差】**

A. 標尺の目盛誤差

B. 標尺の零点誤差

C. 三脚の沈下による誤差

D. 視準点誤差

E. 球差

**【対策法】**

ア. 往観測と復観測で標尺を入れ替える。

イ. 観測点数を偶数にする。

ウ. 三脚の二本を視準線に平行に設置し、そのうちの一本を常に同一標尺に向ける。

エ. 2本の標尺を交互にレベルを置き、レベルと前視標尺及び後視標尺との距離を等しくする。

オ. 後視、前視、前視、後視の順に観測する。

### 解説

**正解：5**

| **誤差**            | **対策法** | **理由**                                 |
| ----------------- | ------- | -------------------------------------- |
| **A. 標尺の目盛誤差**    | **ア**   | 2本の標尺の特性を平均化するため。                      |
| **B. 標尺の零点誤差**    | **イ**   | 偶数点にすることで、出発点と到着点で同じ標尺（またはペア）になり相殺される。 |
| **C. 三脚の沈下による誤差** | **オ**   | 「後・前・前・後」の順で測ることで、時間の経過に伴う沈下量を相殺する。    |
| **D. 視準点誤差**      | **エ**   | レベルの傾きを一定方向に管理するため。                    |
| **E. 球差**         | **エ**   | 前後視の距離を等しくすることで消去。                     |

## 問C：標尺補正と最確値の計算

### 問題

新点Pの標高を求めるため図のような路線で1級水準測量を行い、表1の結果を得た。標尺補正を行った後の新点Pの標高はいくらか。

（※既知点の標高は表2の通り。使用した標尺の標尺改正数は $20^\circ\text{C}$ において $+8\mu\text{m/m}$、膨張係数は $+1.5 \times 10^{-6}/^\circ\text{C}$ である。観測値の重量は距離の逆数に比例するものとする。）

**表1：観測データ**

| 区間 | 観測比高 | 距離 | 平均気温 |

| :--- | :--- | :--- | :--- |

| A → P | 10.6612m | 2km | $20^\circ\text{C}$ |

| P → B | -24.8210m | 2km | $18^\circ\text{C}$ |

| C → P | 55.4538m | 4km | $15^\circ\text{C}$ |

### 解説

**正解：4**

1. **標尺補正計算**
    
    公式：$\Delta C = (C_0 + (T - T_0) \cdot \alpha) \cdot \Delta H$
    
    - **A→P：** $\Delta C = (8 + (20-20) \times 1.5) \times 10^{-6} \times 10.6612 \approx +0.0001m$
        
        標高 $P = 200.147 + 10.6612 + 0.0001 = 210.8083m$
        
    - **P→B：** $\Delta C = (8 + (18-20) \times 1.5) \times 10^{-6} \times (-24.8210) \approx -0.0001m$
        
        標高 $P = 185.985 - (-24.8210 - 0.0001) = 210.8061m$
        
    - **C→P：** $\Delta C = (8 + (15-20) \times 1.5) \times 10^{-6} \times 55.4538 \approx +0.00003m$
        
        標高 $P = 155.353 + 55.4538 + 0.0000 = 210.8087m$
        
2. **重量平均による最確値**
    
    距離の逆数が重量なので、$P_A : P_B : P_C = 1/2 : 1/2 : 1/4 = 2 : 2 : 1$
    
    $$\text{最確値} = 210.800 + \frac{(0.0083 \times 2) + (0.0061 \times 2) + (0.0087 \times 1)}{2+2+1} = 210.8075... \approx 210.808m$$
