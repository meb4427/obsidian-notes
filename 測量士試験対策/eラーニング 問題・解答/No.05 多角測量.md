# 多角測量 No.05 解答と解説

## 問A：重量を用いた観測値の平均 #99NG 

### 問題

地点A、B間の距離を同じ光波測距儀を用いて3日間測定し、表5-1のような結果を得た。測定したA、B間の距離の最確値はいくらか。

（※気象補正、器械定数補正、反射鏡定数補正は行われているものとする）

**表5-1**

| 観測日 | 観測値の平均 | 平均値の標準偏差 |

| :--- | :--- | :--- |

| 1日目 | 420.244m | 10mm |

| 2日目 | 420.248m | 10mm |

| 3日目 | 420.246m | 20mm |

### 解説

**正解：4**

観測精度（標準偏差）が異なる場合、精度の高いものに大きな「重量（重み）」をかけて平均を計算します。

重量 $P$ と標準偏差 $m$ の間には以下の関係があります。

$$P = \frac{1}{m^2}$$

1. **各日の重量を求める**
    
    - 1日目（$P_1$）：$\frac{1}{10^2}$
        
    - 2日目（$P_2$）：$\frac{1}{10^2}$
        
    - 3日目（$P_3$）：$\frac{1}{20^2}$
        
2. **重量比を整列する**
    
    - $P_1 : P_2 : P_3 = \frac{1}{100} : \frac{1}{100} : \frac{1}{400} = 4 : 4 : 1$
        
3. **最確値を計算する（小数点以下のみで計算）**
    
    - $\text{最確値} = 420.240m + \frac{(0.004 \times 4) + (0.008 \times 4) + (0.006 \times 1)}{4+4+1}$
        
    - $= 420.240m + 0.006m = 420.246m$

## 問B：基準面上の距離の計算 #99NG 

### 問題

図5-1のように、標高24.00mの点Aと標高84.71mの点Bの間の距離及び高低角の観測を行い、表5-2の結果を得た。基準面上の距離 $S$ はいくらか。

（※地球平均半径 $R = 6,370km$、ジオイド高 $N_g = 43.00m$）

**表5-2**

| 項目 | 観測値 |

| :--- | :--- |

| 測定距離 $D$ | 1,160.000m |

| 高低角 $\alpha_1$ | $2^\circ 59' 24''$ |

| 高低角 $\alpha_2$ | $-3^\circ 00' 36''$ |

| 器械高・目標高 | $i_1, f_1 = 1.400m / i_2, f_2 = 1.500m$ |

### 解説

**正解：2**

準則の付録6に基づき、以下の式で計算します。

$$S = D \cos\left(\frac{\alpha_1 - \alpha_2}{2}\right) \frac{R}{R + \frac{H_1 + H_2}{2} + N_g}$$

1. **平均高低角の計算**
    
    - $\frac{\alpha_1 - \alpha_2}{2} = \frac{2^\circ 59' 24'' - (-3^\circ 00' 36'')}{2} = 3^\circ$
        
2. **平均標高の計算**
    
    - $\frac{H_1 + H_2}{2} = \frac{(24.00+1.400) + (84.71+1.500)}{2} = 55.805m$
        
3. **代入して計算**
    
    - $S = 1,160.000 \times \cos 3^\circ \times \frac{6,370,000}{6,370,000 + 55.805 + 43.00}$
        
    - $S = 1,160.000 \times 0.99863 \times 0.999984 = 1,158.393m$

## 問C：相互偏心補正計算 #99NG 

### 問題

障害物があったため、図5-2のようにA、B両点で偏心観測を行った。基準点A、B間の球面距離 $S$ はいくらか。

**表5-3（抜粋）**

- $S_1 = 1,000.000m$ / $e_1 = 40.000m$ / $e_2 = 30.000m$
    
- 偏心角：$\alpha_1 = 330^\circ 00' 00''$ / $\alpha_2 = 300^\circ 00' 00''$
    

### 解説

**正解：4**

測線の両側で偏心が行われる「相互偏心」の問題です。三平方の定理を活用するために、基準線 $S_1$ に平行な成分と垂直な成分に分解して考えます。

1. **垂直成分（AE）と水平成分（EB）の計算**
    
    - $a_1 = 360^\circ - 330^\circ = 30^\circ$
        
    - $a_2 = 360^\circ - 300^\circ = 60^\circ$
        
    - $AE = e_1 \sin a_1 + e_2 \sin a_2 = 40.000 \times 0.5 + 30.000 \times 0.866 = 45.981m$
        
    - $EB = S_1 - e_1 \cos a_1 - e_2 \cos a_2 = 1,000.000 - 34.641 - 15.000 = 950.359m$
        
2. **三平方の定理で真の距離 $S$ を求める**
    
    - $S = \sqrt{AE^2 + EB^2} = \sqrt{45.981^2 + 950.359^2} \approx 951.471m$
